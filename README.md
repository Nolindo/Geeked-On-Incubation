# Gallery Management System

## Overview
The Gallery Management System is a web-based application developed using ASP.NET MVC that allows users to browse, upload, and manage image galleries. Images are organized into categories, making it easy to search and view different collections such as Animals, Nature, Sports, Weddings, and Graduations.

This project was developed as part of my Diploma in Information and Communication Technology (Application Development) at Durban University of Technology Incubation

---

## Features

- View image gallery
- Browse images by category
- Upload new images
- Edit image details
- Delete images
- Manage categories
- Responsive user interface
- Image preview with Lightbox
